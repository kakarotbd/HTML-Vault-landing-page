What Is HTML Vault?

HTML Vault is ELECSY's proprietary DRM system — the world's first encrypted delivery platform for Electron, HTML, and Python desktop apps. Your source code never touches the client machine. It stays encrypted in our secure server, decoded only at runtime when the app runs on an approved device.

01

Triple-Layer Encryption

AES-256-GCM × 2 plus Homophonic Custom ENC. Three completely separate keys stored server-side. Mathematically impossible to crack without all three.

02

Hardware Device Lock

Each EXE is bound to approved machines via BIOS serial, UUID, MAC address, CPU, and RAM fingerprinting. Copied EXE files don't work anywhere else.

03

Server-Side Decryption

Raw source code never reaches the client machine. Decryption happens inside ELECSY's Cloudflare Worker. The client only sees the rendered output.

04

Instant Updates

Push updates from your admin panel. All running apps silently receive changes within 60 seconds. No new EXE. No client visits. Just save and done.

05

Remote Block & Control

Non-paying client? Block them in one click from the admin panel. Their app stops working immediately. No calls, no visits, no stress.

06

Client Editor Portal

Give clients a secure private link to edit only their own content. Changes save directly to the encrypted backend. You stay in full control.

07

HMAC Token Auth

Every session uses a cryptographically signed one-time token. Replay attacks, brute force, and token theft are all blocked by design.

08

Backend Popup System

Send force popups, one-time notices, or repeat interval messages to your clients — all from admin panel without touching the EXE.

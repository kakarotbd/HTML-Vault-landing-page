SECURITY

Why It's Unbreakable

Every layer is designed to stop attackers cold — even with physical access to the machine.

🔑
Keys Never Leave Server
All three encryption keys live in Cloudflare environment variables. Never exposed in responses or logs.
⏱️
Replay Attack Blocked
HMAC-signed nonce + timestamp tokens. One-time use only. 60-second expiry window.
💻
Hardware Fingerprinting
BIOS serial, UUID, MAC, CPU, RAM — SHA256 hashed. Impossible to spoof without exact values.
🚫
Raw Code Never Exposed
Decryption happens server-side. Client receives rendered output only. Source stays protected.
